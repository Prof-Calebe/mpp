Computational Environment


The Personal Computer

Each team will have one PC to build the solutions. This PC has the following configuration:

  • Dual-Core 3.0 GHz
  • 4.0 Gb RAM
  • Debian OS 5.0
  • GCC 4.2 (with OpenMP)
  • Open-MPI 1.3.3

The Test Machine

All teams have access to a mini-cluster with 4 computers exactly equals to the Judge Machine. Only the network differs. Special thanks to SGI and Intel.

  • 2xQuad-Core XEON 2.0 GHz
  • 8.0 Gb RAM
  • SUSE Linux Enterprise Server 10
  • Inter Compiler (C/C++) 9.0
  • Intel Cluster Toolkit with MPI 3.0
  • Gigabit ethernet

its sound...

The Judge Machine

The target machine, which will be used to execute the team's solution, is a cluster with 10 computers. Each computer has 8 processors.

  • 2xQuad-Core XEON 2.0 GHz
  • 8.0 Gb RAM
  • SUSE Linux Enterprise Server 10
  • Inter Compiler (C/C++) 9.0
  • Intel Cluster Toolkit with MPI 3.0
  • InfiniBand
