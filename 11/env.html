Computational Environment


The Test and Judge Machine

All teams have access to the target machine. It is a cluster named Enterprise3 with 30 computers (each computer has 4 cores). It is necessary to use the SGE (Sun Grid Engine) to submit a job. The SGE controls the job queue ensuring that time sharing will not occurs. Special thanks to LCAD/UFES. More information about the environment can be found at the LCAD wiki page.

  • Intel Core 2 Quad CPU 2.40GHz
  • 2.0 Gb RAM
  • Fedora Core 8 32-bits
  • Gigabit Ethernet
  • gcc 4.1.2
  • LAM-MPI 7.1.4
