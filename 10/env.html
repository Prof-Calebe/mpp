Computational Environment


The Personal Computer

Each team will have one PC to build the solutions. This PC has the following configuration:

  • IBM Xeon 3200 serie
  • CentOS 5.4

The Test and Judge Machine

All teams have access to the target machine. It is a cluster with 30 computers (each computer has 8 cores) but each user can have at most 48 processors. It is necessary to use the SGE (Sun Grid Engine) to submit a job. The SGE controls the job queue ensuring that time sharing will not occurs. Special thanks to LNCC and SGI. More information about the environment can be found at the LNCC/Altix-XE cluster.

  • 2xQuad-Core Intel Xeon E5520 2.27GHz
  • 24.0 Gb RAM
  • CentOS 4.5
  • InfiniBand
