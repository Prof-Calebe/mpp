The Test and Judge Machine

All local teams have access to an Intel Xeon and Intel Xeon Phi machine:

  • 2 x Intel(R) Xeon(R) CPU E5-2670 0 @ 2.60GHz (total 16 cores / 32 threads)
  • 64Gb RAM
  • 1 x Intel Xeon Phi 5110P (60 cores)
  • 1 x Intel Xeon Phi 3120A (57 cores)
  • CentOS 7.1
  • Intel(R) C/C++ Compiler 64 15.0.3.187 Build 20150407
  • Intel(R) MPI Library 5.0 Update 3
  • some material
  • GCC C/C++ Compiler 4.8.3 Build 20140911
